# CodeAlpha Quote Generator App

**Domain:** Flutter App Development Internship **Intern:** Maryam Javed **Organization:** CodeAlpha

## 📱 Project Overview
This Flutter application is developed as part of the CodeAlpha Flutter Development Internship. The app generates random inspirational and motivational quotes to uplift users' mood and provide daily inspiration.

## ✨ Features
- Displays random quotes on button tap
- Clean and modern Material UI design 
- Share functionality to share quotes with friends
- Responsive design for all screen sizes
- Smooth animations and transitions

## 🛠️ Tech Stack
- **Framework:** Flutter
- **Language:** Dart  
- **IDE:** Android Studio / VS Code
- **State Management:** setState

## 🚀 How to Run
1. Clone this repository: `git clone https://github.com/maryamjaved416-tech/Codealpha_tasks.git`
2. Navigate to project: `cd Codealpha_tasks/CodeAlpha_QuoteGenerator`
3. Install dependencies: `flutter pub get`
4. Run the app: `flutter run`
